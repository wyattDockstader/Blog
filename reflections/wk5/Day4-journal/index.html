What is a virtual property? additional fields given to a model either manually or through defined functionality

When might you use a virtual property? Anytime you want to combine multiple document values and return them at the same time without having to manually append the values in the request.

How do you search by a virtual properties value? .virtual("virtual property name").get(function())
